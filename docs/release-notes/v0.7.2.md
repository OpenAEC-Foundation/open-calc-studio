# v0.7.2 — Public release on clean repo

**Uitgebracht:** 2026-05-02

## Schone publieke release

Open Calc Studio is verhuisd naar een schone publieke repo: [github.com/OpenAEC-Foundation/open-calc-studio](https://github.com/OpenAEC-Foundation/open-calc-studio).

Geen klantdata, geen vendor-logos, geen interne documenten in de history.

## Defaults aangepast

- **Taal:** Nederlands (was: auto-detect)
- **Thema:** Light (was: 'default')

Bij eerste start wordt de UI standaard in het Nederlands en lichte thema geladen. Te wijzigen via Instellingen.

## Bevat (sinds v0.7.0)

- Persistent left start sidebar (release notes + OpenAEC link + voorbeeldbegroting)
- Voorbeeldbegroting auto-load bij eerste start
- Brand-neutrale logo's (geen 3e-partij logos in installer)
- Live staart-cascade in Bouw 1 PDF (Rust-side berekening, altijd correct)
- Spreadsheet copy/paste/Delete via sneltoetsen (focus-fix)
- Sticky kolomheaders in spreadsheet (CSS zoom ipv transform)
- Tauri opener-permissions voor PDF auto-open

## Compat
Alle v0.7.x bestanden openen normaal.
