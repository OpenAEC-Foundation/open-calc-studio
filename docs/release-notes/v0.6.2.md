# v0.6.2 — Staart-migratie hotfix

**Uitgebracht:** 2026-05-01

## Hotfix
v0.6.1 introduceerde live staart-berekening uit items[]. Maar legacy WpCalc-imported bestanden hadden geen staart_* items in items[] — staart zat alleen in `schedule.staartRows` cache. Resultaat: rapport had geen staart meer (0 ipv totaal).

**Fix:** bij bestand-load worden staart_* items automatisch geïnjecteerd als ze ontbreken. Percentages komen uit de gecachete staartRows (indien aanwezig), anders standaard NL bouw waarden.

WpCalc importer maakt nu ook direct staart_* items.

## Compatibiliteit
Bestaande v2.1 bestanden openen normaal. Bij eerste save worden de staart-items mee opgeslagen.
