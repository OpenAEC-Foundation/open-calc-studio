# v0.7.1 — Public testfase release

**Uitgebracht:** 2026-05-01

## Public release voor testfase

Open Calc Studio is nu beschikbaar voor publieke testfase. Een open-source begrotingsprogramma voor de Nederlandse bouw — vrij te downloaden, vrij te gebruiken, vrij om aan bij te dragen.

## Bug fixes

### 🔥 Staart in PDF-rapport (kritiek)
De staart-cascade (AK over OA → ABK → garanties → W&P → kostprijs → risico → winst → verzekering → BTW → totaal incl. BTW) wordt nu **server-side in Rust berekend** uit live items. Geen afhankelijkheid meer van TS-gecachete waarden.

Resultaat: rapport-totalen zijn altijd identiek aan grid-totalen, ook na wijzigingen in items, percentages, of staart-volgorde.

### 🔧 Andere fixes
- `addDocument` triggert nu `recalculateItems` zodat staartItemBreakdown gevuld wordt bij file-open
- Spreadsheet copy/paste/cut/Delete: container krijgt focus na cel-klik, sneltoetsen werken nu correct
- PDF-export auto-open: Tauri opener-permissions toegevoegd voor file paths
- Release notes panel valt terug op bundled lijst als GitHub onbereikbaar
- OpenAEC link wijst naar [www.open-aec.com/open-calc-studio](https://www.open-aec.com/open-calc-studio/)

## Geen 3e-partij logo's meer in de installer

Geen vendor-logo's gebundeld in de release. PDF-rapporten gebruiken alleen logo's die de gebruiker zelf uploadt via **Bedrijfsgegevens → Logo links/rechts**. Dit maakt OCS volledig brand-neutraal.

Wil je toch je eigen logo? Upload een PNG via de Bedrijfsgegevens-panel (links/rechts).

## Auto-open voorbeeldbegroting

Bij eerste start van de app wordt automatisch de voorbeeldbegroting geladen — direct context, geen leeg startscherm. Sluit het tabblad om door te gaan met je eigen werk.

## Compat
Alle v0.7.0 bestanden openen normaal. Geen migratie nodig.
