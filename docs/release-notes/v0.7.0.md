# v0.7.0 — Persistent Start Sidebar + critical staart fix

**Uitgebracht:** 2026-05-01

## Nieuw

### 🎉 Persistent Start sidebar
Het welkomscherm is nu een **altijd-zichtbare linker sidebar**, ook met begrotingen open. Bevat:
- Snelkoppelingen: **Nieuw**, **Openen**, **Voorbeeldbegroting**
- **Recente bestanden** lijst
- **Live release notes** uit GitHub (6h cache)
- Link naar [www.open-aec.com](https://www.open-aec.com/)

Sidebar te collapsen naar een lipje aan de linkerkant. Voorkeur opgeslagen in localStorage.

### 📂 Uitgebreide voorbeeldbegroting
Sample begroting nu 21 items, kostprijs ≈ €44K, 6 hoofdstukken (grondwerk, betonwerk, ruwbouw timmerwerk, isolatie, schilderwerk, elektrotechnische installaties).

### 🔧 Spreadsheet auto-open
Klikken op "Spreadsheet" tab opent nu automatisch een sheet als er nog geen bestaat.

### 🎨 Rapport-logo preset
Schedule heeft nu `reportLogoPreset` (Standaard / Eigen logo). "Eigen logo" werkt via CompanyInfo upload.

## Critical fix

### 🔥 Staart in PDF rapport (kritiek)
v0.6.1+ introduceerde live staart-berekening. Maar `setItems()` in costItemsSlice riep `recalculateItems` niet aan → staart-items kwamen wel in items[] maar zonder breakdown → Bouw 1 PDF toonde lege staart-tabel.

**Fix:** `setItems(items)` triggert nu `recalculateItems(items)`. Bij elk file-open wordt staart breakdown direct correct gevuld.

## Cleanup
- "Offerte" verwijderd uit Rapportage-views (bestaat als eigen ribbon-tab)
- Experimentele tabs (PDF / 3D / Offerte) verborgen in zowel dev als release builds

## Compat
