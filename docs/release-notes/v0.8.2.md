# Open Calc Studio v0.8.2

**Uitgebracht:** 2026-06-12

## Wijziging

OpenAEC-integraties (login, cloud-opslag, AI-via-account) zijn nu een
build-time feature flag (`VITE_OPENAEC_ENABLED`) en standaard **uit** in
publieke builds. De publieke release toont dus geen OpenAEC-koppelingen
meer in de titelbalk, het Bestand-menu of de calculatieassistent — alleen
zichtbaar in builds die expliciet met de flag aan worden gemaakt.

De ingebouwde assistent blijft beschikbaar via een eigen Anthropic
API-sleutel of de offline antwoorden over de geopende begroting.
