# v0.7.4 — Uren naar rato, IBIS-TRAD import, volledige API

**Uitgebracht:** 2026-05-02

## Nieuw

### ⏱️ Uren naar rato terug-rekenen (3 niveaus)
Pas een totaal aantal uren aan en de onderliggende productienormen worden proportioneel herrekend:
- **Per hoofdstuk** — klik op de Uren-cel van een hoofdstukrij in de WPCalc-weergave; alle regel-normen in dat hoofdstuk (ook geneste posten/sub-hoofdstukken) schalen naar rato
- **Per tariefgroep** (A/B/C) — in het Uren-overzicht paneel
- **Grand-totaal** — de totaalrij in het Uren-overzicht (verdeelt over alle groepen)

Edge-cases: huidig totaal 0 → gelijkmatig verdeeld; doel 0 → normen op nul.

### 📥 IBIS-TRAD import (.xtb)
Importeer begrotingen uit IBIS-TRAD `.xtb` SQLite-bestanden via **Bestand → Importeren → IBIS-TRAD (.xtb)**. Leest hoofdstukken, posten, kostenposten en middelen (arbeid/materiaal/materieel/onderaanneming).

### 🔗 File-associaties + drag-and-drop
- Dubbelklik op `.ifcCalc` of `.calc` in Verkenner opent OCS
- Sleep een bestand op het OCS-venster → opent direct

### 🤖 Volledige MCP + REST API
De MCP-server (voor AI-agents) en een nieuwe lokale REST API (127.0.0.1:9742) dekken nu de volledige app-functionaliteit:
- Spreadsheet-bladen (toevoegen/hernoemen/verwijderen, cellen lezen/schrijven, batch)
- Begrotingsvarianten (branches: aanmaken, schakelen, verwijderen)
- Bedrijfs- en projectgegevens (lezen/bijwerken)
- Middelenbibliotheek (CRUD)
- Boom-operaties (move, indent/outdent, collapse)
- Import: CUF-XML, TRADXML, RSX, ZSX, NSX, .xtb, .calc
- Documenten (tabs: lijst, schakelen, toevoegen/sluiten)
